# Aven (AVY)
Aven is a decentralized asset developed by Stackhouse-Software on the Solana (SPL) standard.
The project provides utility for decentralized applications (dApps) across the Stackhouse-Software product line.

Facilitating secure customer interactions within a closed ecosystem. No public LP.

By maintaining a closed software environment,
the project protects users from the inference of bad actors and external influences.

This architecture ensures the asset is utilized outside of bot-driven systems.
Effectively circumventing vulnerabilities commonly found in incentivized public programs.

Key Features

>    Decentralized Utility: Native integration across all Stackhouse-Software dApps.
>    Closed Ecosystem: Enhanced security through a controlled software environment.
>    Bot Mitigation: Designed to prioritize human interaction over automated inference.
>    Risk Circumvention: Built to bypass the structural flaws of public incentive models.
